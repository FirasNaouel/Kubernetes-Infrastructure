# Kubernetes Infrastructure

This repository defines the Kubernetes infrastructure used to deploy a
containerized static website running on a Kubernetes cluster.

## Architecture

Cloudflare Tunnel → NGINX Ingress → Kubernetes Services → Pods

## Scope
- Kubernetes manifests
- Ingress configuration
- Monitoring and security policies

Application source code and Docker image builds are [here](https://github.com/FirasNaouel/Portfolio-website)
